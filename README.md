# SberTech ML Course: классические алгоритмы машинного обучения

Этот репозиторий содержит домашние задания, выполненные в рамках курса по машинному обучению на кафедре SberTech.

Основной фокус заданий — реализация классических алгоритмов машинного обучения с нуля, понимание математической основы моделей, функций потерь, градиентов, оптимизации и построения пайплайнов для задач классификации.

## Обзор

В репозитории представлены три домашних задания:

- **HW1 — Softmax Regression**
- **HW2 — Support Vector Machines**
- **HW3 — Ensembles**
